## Filter `amp_skip_post`

```php
apply_filters( 'amp_skip_post', $skipped, $post_id, $post );
```

Filters whether to skip the post from AMP.

### Arguments

* `bool $skipped` - Skipped.
* `int $post_id` - Post ID.
* `\WP_Post $post` - Post.
